"Arduino Project: MP3"

Arduino Github

Using C++ for the programming language through Arduino, my partner and I created a program simulating an MP3 player.

The video shows how the mechanism works after the code was completed.

When the program begins, the music starts playing through the buzzer and the red light blinks to the music.

When the user presses the button, the code cycles to the next song and displays the title of the song on the screen.

The user can cycle back and forth between the two songs as many times as they like.
