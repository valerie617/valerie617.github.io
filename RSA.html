"RSA Project: Encryption and Decryption"

RSA Github

Using Canopy for the IDE, Python for the programming language, and Tkinter as a GUI, my partner and I created a program that can encrypt and decrypt messages using a public and private key. The public key was available to our peers so they could encrypt messages and send them to Jacob and me. Then, with our private key embedded in the program, we decrypted the messages. In response, we also inserted other groups' public key into our program, encrypted a unique message and sent it to them to decrypt. A major problem we encountered is the randomness of the program. Sometimes it would work, but other times we would recieve gibberish. Eventually, we realized that copying and pasting the message has to be done carefully so as not to copy anything that was not part of the message. Overall, it was a fun project with interesting concepts, and it further explored the benefits of Tkinter.
